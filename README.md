# Biometric Voting System

A secure voting system built using Django that verifies voters using Face Recognition and Iris Detection before allowing them to vote.

## Features

- Voter Registration
- Face Recognition Verification
- Iris Detection Verification
- Secure Voting System
- Vote Result Display
- Admin Panel for Management

## Technologies Used

- Python
- Django
- OpenCV
- Face Recognition
- NumPy
- SQLite Database
- HTML, CSS, JavaScript
